### Certisfy SDK For Javascript

This is the core implementation SDK of Certisfy for Javascript.

The Certisfy SDK consists of the following key components:

1. [signer](https://github.com/CipherTrustee/certisfy-js/tree/d8af6606095ab4bc80903622f8ac6a0f72835b20/docs/signer)
2. [verifier](https://github.com/CipherTrustee/certisfy-js/tree/d8af6606095ab4bc80903622f8ac6a0f72835b20/docs/verifier)
3. [cert-gen](https://github.com/CipherTrustee/certisfy-js/tree/d8af6606095ab4bc80903622f8ac6a0f72835b20/docs/cert-gen)
4. [core-api](https://github.com/CipherTrustee/certisfy-js/tree/d8af6606095ab4bc80903622f8ac6a0f72835b20/docs/core-api)
