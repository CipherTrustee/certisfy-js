### Certisfy SDK For Javascript

This is the core implementation SDK of Certisfy for Javascript.

The Certisfy SDK consists of the following key components:

1. [signer](https://github.com/CipherTrustee/certisfy-js/tree/6f58909632db565429630bdfefe53a7fb51599c5/docs/signer)\
2. [verifier](https://github.com/CipherTrustee/certisfy-js/tree/6f58909632db565429630bdfefe53a7fb51599c5/docs/verifier)\
3. [cert-gen](https://github.com/CipherTrustee/certisfy-js/tree/6f58909632db565429630bdfefe53a7fb51599c5/docs/cert-gen)\
4. [core-api](https://github.com/CipherTrustee/certisfy-js/tree/6f58909632db565429630bdfefe53a7fb51599c5/docs/core-api)\
